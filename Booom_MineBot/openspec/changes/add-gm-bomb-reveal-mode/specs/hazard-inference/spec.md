## ADDED Requirements

### Requirement: GM 模式必须允许临时显示隐藏炸药位置
系统 SHALL 提供一个默认关闭的 GM 显雷模式，用于在运行时显示当前仍然存在于岩壁中的炸药位置。该模式 MUST 只影响表现层可视化，MUST NOT 改变炸药分布、探测反馈、标记状态、机器人目标选择或爆炸结算。

#### Scenario: 正常模式下炸药仍然隐藏
- **WHEN** 玩家处于默认的正常模式，且某块岩壁仍然含有未触发炸药
- **THEN** 该岩壁不会因为新增 GM 能力而在普通表现层中直接暴露炸药位置

#### Scenario: 开启 GM 模式后显示当前炸药格
- **WHEN** 开发者在运行时开启 GM 显雷模式
- **THEN** 所有当前仍满足“岩壁且含有炸药”的格子都会显示额外的调试 overlay

#### Scenario: 关闭 GM 模式后立即清空显雷 overlay
- **WHEN** 开发者在运行时关闭 GM 显雷模式
- **THEN** 调试 overlay 会立即消失，并恢复到正常隐藏炸药的表现
